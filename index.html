Experiencia

Com parceria da FIAP com a empresa B2W, desenvolvemos um sistema que auxilia no seu processo seletivo. Eu fiquei responsavel pelo back-end, que foi feito em java, usando orientação a objetos, Design Patterns para conexão com o banco de dados e Java-web para front-end.

O projeto em si consiste em uma plataforma que o usuario pode usar para poder ficar por dentro das vagas de emprego disponiveis e se candidatar. Os administradores podem criar novas vagas e alertar os usuarios sobre eventos que a empresa está oferecendo.
